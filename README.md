# 研究設計書
## 国会発言における政党別反応の差異分析：台湾有事関連イベントを事例として

> **作成日：** 2025年  
> **所属：** 情報学部 人工知能専攻  
> **ステータス：** 進行中

---

## 1. この研究は何をするのか

### 一言で言うと
> 「ペロシ訪台（2022年8月）という外交イベントの前後で、日本の政党ごとに国会発言がどう変わったかをテキストデータで示す」

### 何がわかるのか
- 政党によって外交イベントへの**反応速度・発言量・使う言葉が違う**ことを定量的に示せる
- 「なんとなく自民と立憲は違う」という印象論を**データで証明**できる
- 将来的な炎上予測モデルの**基盤データ**になる

---

## 2. なぜこの研究をやるのか

### 背景
- 政治家の国会発言がSNSで拡散・炎上する事例が増えている
- しかし「どの政党が・どんな時に・どんな発言をするか」は定量的に明らかになっていない
- 国会議事録は無料・無制限で取得できるにもかかわらず、NLPで分析した研究は少ない

### 研究の位置づけ
```
既存研究：SNSの炎上分析（Twitterデータ中心）
　　↓　ここが空白地帯
本研究：炎上の「源泉」である国会発言そのものを分析
```

---

## 3. 研究のロードマップ

### Phase 1：7月ポスター発表まで
**目標：「政党によって反応が違う」を頻度分析で示す**

```
国会議事録APIでデータ取得
　　↓
「台湾」「中国」「安全保障」の
キーワード頻度を政党別・時系列で集計
　　↓
ペロシ訪台前後3ヶ月の比較グラフを作成
　　↓
Streamlit + Hugging Face Spacesでデプロイ
```

**この段階でわかること**
- 政党ごとの反応量の違い
- 反応のタイミングの違い
- よく使われるキーワードの政党差

---

### Phase 2：卒論前半
**目標：発言の「過激さ」を数値化する**

```
Phase 1のデータを使って
　　↓
BERTで発言ごとに
攻撃性・感情強度をスコアリング
　　↓
「過激さスコア」の時系列変化を可視化
政党ごとに比較する
```

**この段階でわかること**
- イベント前後で過激さがどう変化したか
- 過激さの変化パターンに政党差があるか

---

### Phase 3：卒論後半（うまくいけば）
**目標：炎上パターンの特定**

```
過激さスコアが高い発言を抽出
　　↓
共通する言語的特徴を分析
（どんな言葉・構造が含まれるか）
　　↓
「炎上しやすい発言パターン」を定義する
```

---

### Phase 4：発展（卒論以降）
**目標：炎上予測モデルへ**

```
Phase 3で特定したパターンを使って
任意の議事録テキストを入力すると
炎上リスクスコアが出るモデルを構築
```

---

## 4. データと技術

### データソース
| データ | 取得方法 | コスト |
|---|---|---|
| 国会議事録 | 国会会議録検索システムAPI | 無料・無制限 |

### 分析対象
| 項目 | 内容 |
|---|---|
| 対象イベント | ペロシ訪台（2022年8月） |
| 分析期間 | 2022年5月〜2022年11月（前後3ヶ月） |
| 対象政党 | 自由民主党・立憲民主党・日本維新の会 |
| キーワード（初期）| 台湾・中国・安全保障・有事 |

### 技術スタック
```
データ収集：国会API（requests）
前処理：MeCab + ipadic-NEologd / GiNZA
分析Phase1：キーワード頻度・共起分析
分析Phase2：tohoku-nlp/bert-base-japanese
可視化：matplotlib / seaborn / plotly
アプリ：Streamlit
デプロイ：Hugging Face Spaces
管理：GitHub
```

---

## 5. 現時点での限界（正直に書いておく）

- 「炎上した」かどうかのラベルはPhase 1では使わない
- 「過激さ」の定義はBERTのスコアで代替するが、主観が入る余地がある
- 対象政党は3党に絞るため全体像は反映しない
- Phase 3以降は就活・時間次第でスコープを調整する
